# Whisper docker setup

This is a simple docker setup with an FastAPI proxy in front to add basic authentication.

## Setup

```shell
idc build
idc up -d
```

See [http://whisper.local.itkdev.dk/docs](http://whisper.local.itkdev.dk/docs)
